# Отчёт по нагрузочному тестированию

## Цель

Проверить, что endpoint поиска выдерживает 50 одновременных запросов и возвращает ответы без ошибок.

## Сценарий

Перед запуском теста через API был загружен документ `valid-lecture.docx`, после чего выполнялись параллельные запросы:

```text
GET /api/v1/search?q=EPrac&page=1&page_size=10
```

Для запуска использовался скрипт:

```bash
python backend/tests/performance/search_load_test.py --base-url http://localhost:8001 --query EPrac --users 50
```
